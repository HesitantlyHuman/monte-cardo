# Monte Cardo
Monte Cardo is a Monte Carlo Tree Search (MCTS) based game tree evaluator (read AI) for ladder shedding games like [Scum](https://en.wikipedia.org/wiki/President_(card_game)) or [The Great Dalmuti](https://en.wikipedia.org/wiki/The_Great_Dalmuti).

## Performance Testing
Monte Cardo uses `criterion.rs` to do performance and regression testing. To start the benchmarks, simply run the following:

```bash
cargo bench
```
